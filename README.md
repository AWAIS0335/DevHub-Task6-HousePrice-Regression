# Task 6: House Price Prediction 🏡💰

This repository contains my solution for Task 6 of the AI/ML Engineering Internship at DevelopersHub Corporation.

**Objective:** Predict house prices using property features such as size, bedrooms, and location.
**Dataset:** California Housing Dataset (Scikit-Learn).
**Model Used:** Gradient Boosting Regressor.
**Evaluation Metrics:** Mean Absolute Error (MAE) and Root Mean Squared Error (RMSE).
**Tools & Libraries:** Python, Pandas, Scikit-Learn, Matplotlib.
